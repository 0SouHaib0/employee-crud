# employee-crud(Spring Boot + PostgreSQL)

## Tech Stack

- Java 17
- Spring Boot 3.5.3
- Spring Data JPA
- PostgreSQL
- Maven

## Features

- Full CRUD operations for Employees
- Input validation using Jakarta Validation
- Clean architecture and RESTful principles
